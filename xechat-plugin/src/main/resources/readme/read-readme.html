阅读功能说明

阅读功能支持添加本地书籍和Legado书籍

本地书籍

本地书籍只支持txt文件，点击开始阅读后在书架界面点添加然后按步骤完成添加即可，插件会自动生成章节目录（可自定义匹配章节的正则表达式），无法识别章节目录的文件不能添加。

Legado书籍

Legado（阅读3.0）是一款开源的手机端小说阅读app，功能十分强大。本插件支持添加Legado书架中的书籍，可双向同步阅读进度。在添加Legado书籍之前请先在阅读app的设置中打开Web 服务，然后在插件设置中填写Legado服务的IP即可。

模式切换

可以在阅读界面进行模式切换，目前有简单和困难两种模式。
  • 简单模式：在阅读面板中显示内容。
  • 困难模式：在IDEA下方的状态栏显示内容，如果内容被其他消息覆盖，重新翻页即可。

快捷键

所有快捷键均可自定义设置，快捷为全局快捷键，简单模式和困难模式均可使用。
  • 上一页：上一页，默认快捷键[Ctrl+↑]。
  • 下一页：下一页，默认快捷键[Ctrl+↓]。
  • 老板键：一键隐藏小说内容，默认快捷键[Shift+↓]。
  • 恢复键：恢复隐藏的小说内容，默认快捷键[Shift+↑]。
